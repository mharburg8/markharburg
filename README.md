# Mark Harburg - Personal Website

Professional portfolio website for Mark Harburg, Solutions Architect & Sales Engineer.

## Deployment

This site is designed to be deployed on **Cloudflare Pages** or **GitHub Pages**.

### Cloudflare Pages Deployment
1. Push this repository to GitHub
2. Log into Cloudflare Dashboard → Pages
3. Create a new project → Connect to Git
4. Select this repository
5. Build settings:
   - Build command: (leave empty)
   - Build output directory: `/`
6. Deploy

### GitHub Pages Deployment
1. Push this repository to GitHub
2. Go to Settings → Pages
3. Source: Deploy from a branch
4. Branch: `main` / `root`
5. Save
